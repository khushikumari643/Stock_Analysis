# Stock_Analysis
This project is an interactive stock market analysis dashboard that compares stock price trends and company revenue for Netflix and GameStop. Data is extracted using Python libraries and web scraping techniques, then analyzed to identify patterns and relationships between financial performance and market behavior.

The project uses Plotly to create code-based interactive visualizations and Power BI to build a business-oriented dashboard. This combination enables both technical analysis and professional data presentation.

The analysis highlights how stock prices may not always reflect company fundamentals, particularly in volatile cases like GameStop, while companies like Netflix show trends more aligned with revenue growth.
